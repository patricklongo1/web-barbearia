###### Para executar:
1. clonar o diretorio: api-barbearia
2. subir via docker: redis, mongodb, database (Postgres)
3. run: yarn
4. run: yarn dev
5. clonar este diretorio: web-barbearia
6. run: yarn
7. run: yarn start
